# 专利自由实施（FTO）初筛（中国法）

## 目标

针对拟实施的产品、方法或技术方案，在中国境内进行专利侵权风险初筛，识别高风险专利、评估侵权可能性等级、提出规避设计方向与抗辩策略线索。

**性质声明**：本技能为初筛（triage），不构成正式 FTO 法律意见书。正式 FTO 意见须由专利代理师/律师在完整检索基础上出具并签章。

**保密要求**：FTO 分析可能涉及未上市产品技术细节。执行前确认用户的数据分级、允许使用的处理环境和脱敏要求；涉及国家秘密或被明确禁止输入外部系统的信息时停止处理并提示改用获批环境。

## 可选实践档案

如当前项目存在 `LEGAL_PROFILE.md`，先读取其中与本任务相关的企业背景、审批权限、保密要求和风险偏好，并将其作为补充事实使用。

- 不得因档案缺失而拒绝执行任务。
- 不得把档案中的默认值当作用户本次任务已经确认的事实。
- 档案与用户本次明确指示冲突时，以用户本次指示为准。
- 涉及企业名称、联系人、审批人或预算时，未获确认的一律使用占位符。

## 执行前准备

1. 如存在 `LEGAL_PROFILE.md`，仅读取与本次任务相关且已经确认的字段，例如：
   - 企业技术领域、产品方向与目标应用场景
   - 目标市场（中国大陆 / 含港澳台 / 海外）
   - 外部检索服务商（incoPat / 智慧芽 / Patentics）
   - FTO 项目的审批权限与预算阈值

2. 确认用户已提供或可访问产品技术方案的充分描述，包括：
   - 产品/方法的技术特征清单
   - 工作流程图或系统架构图（如有）
   - 与现有产品的差异说明

## 执行步骤

### Step 1: 技术特征分解

将产品/技术方案分解为可比对的技术特征列表（Feature List）。分解原则：
- 按《最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释》第7条，以权利要求记载的全部技术特征为比对对象
- 区分"必要技术特征"与"可选技术特征"
- 对功能性限定特征单独标注（需结合说明书及附图解释）
- 对环境特征、用途特征单独标注

**示例（环境监测领域）**：
```
F1: 一种大气污染监测方法
F2: 通过多光谱遥感卫星获取目标区域的光学厚度数据
F3: 将光学厚度数据与地面监测站数据进行时空融合
F4: 使用神经网络模型反演 PM2.5 浓度分布
F5: 根据气象预报数据对反演结果进行动态修正
F6: 当预测浓度超过阈值时，向监管平台发送预警信息
```

### Step 2: 检索策略制定与执行

**检索范围**：
- 数据库：中国国家知识产权局公开数据库、incoPat、智慧芽、Patentics
- 专利类型：发明专利（含已授权和审中）、实用新型专利
- 法律状态：有效、审中（注意未公开的在先申请无法检索，存在"撞车"风险）
- 时间范围：申请日前所有公开专利 + 近 3 年审中专利（潜在的在先申请）

**检索要素构建**：
1. **关键词**：技术方案的核心名词、动词、效果词
2. **IPC/CPC 分类号**：根据技术领域确定（如 G01N 15/06 颗粒物测量；G06N 3/08 神经网络）
3. **申请人/发明人**：主要竞争对手、高校/科研院所合作方、离职员工新任职企业
4. **引用/被引用关系**：核心专利的后续改进专利

**输出**：
- 检索式（供 incoPat/智慧芽 直接执行）
- 初筛专利清单（按相关度排序，Top 10-20）

### Step 3: 高风险专利识别

对初筛专利进行权利要求解读，识别"高风险专利"：

**高风险判断标准**：
1. **相同技术领域**：IPC 大类相同或相近
2. **相同技术问题**：背景技术描述的问题与本方案一致
3. **技术特征高度重叠**：独立权利要求中的大部分技术特征可被本方案覆盖
4. **有效且稳定**：
   - 法律状态：有效（已缴当年年费）
   - 无无效决定或无效决定维持有效
   - 未涉及权属纠纷
5. **权利人背景**：NPE（非实施实体）/竞争对手/维权活跃企业

**特别关注点**：
- **标准必要专利（SEP）**：若产品涉及环保监测行业标准（如 HJ 系列标准），排查是否存在声明的 SEP
- **相似外观设计**：若产品为硬件设备，同步排查外观设计专利
- **分案申请**：核心专利的分案可能包含更窄但更接近的权利要求
- **实用新型稳定性**：实用新型授权并不等于其在无效程序中当然稳定。应结合申请日适用的审查规则、检索报告、评价报告、现有技术和审查历史逐案判断，不按授权年份直接推定稳定性高低。

### Step 4: 侵权比对初评

对高风险专利进行逐件权利要求对照分析，适用中国专利侵权判定规则：

#### 4.1 全面覆盖原则（司法解释一第7条）

将被控技术方案与专利独立权利要求的全部技术特征逐一比对：
- **字面侵权**：被控方案包含权利要求记载的全部技术特征（相同或等同）
- **变劣技术方案**：故意省略个别必要技术特征，但技术效果变差——中国司法实践中一般不构成侵权（与美国的 doctrine of equivalents 范围不同）
- **多余指定原则**：已被中国司法实践基本摒弃，不得随意将非必要技术特征排除在保护范围之外

#### 4.2 等同原则（司法解释一第8条）

判断被控方案的技术特征是否与权利要求记载的技术特征**以基本相同的手段，实现基本相同的功能，达到基本相同的效果**，且本领域技术人员**无需创造性劳动**就能联想到。

**等同原则的例外/限制**：
1. **禁止反悔**（司法解释二第6条）：申请/无效过程中为获得授权而明确放弃的技术方案，不得在侵权诉讼中通过等同原则重新纳入保护范围
2. **捐献规则**（司法解释二第5条）：仅在说明书或附图中描述而未记载在权利要求中的技术方案，视为捐献给社会公众
3. **特意排除规则**：权利要求明确排除的技术方案，不适用等同原则

#### 4.3 功能性技术特征的特殊解释（司法解释一第4条、司法解释二第8条）

若权利要求中某一技术特征以功能或效果表述：
- 保护范围限定为说明书及附图记载的具体实施方式及其等同实施方式
- 比对时须逐一核查说明书中的具体实施方式

**输出格式**：
```markdown
| 专利号 | 专利名称 | 权利人 | 法律状态 | 风险等级 | 侵权比对结论 |
|:---|:---|:---|:---|:---|:---|
| CN10XXXXXXXX | 一种大气遥感监测方法 | [企业名称] | 有效 | 🔴 高 | 技术特征F2/F3/F4被全面覆盖，F5可能构成等同 |
```

### Step 5: 抗辩策略与规避设计线索

对每件高风险专利，提供以下分析：

#### 5.1 现有技术抗辩（《专利法》第67条）
- 检索被控方案实施日前的现有技术（全球公开）
- 若被控方案的全部技术特征被一篇现有技术公开，则不侵权
- **注意**：现有技术抗辩仅抗辩"不侵权"，不能否定专利权效力（与无效宣告不同）

#### 5.2 先用权抗辩（《专利法》第75条第2项）
- 若企业在专利申请日前已经制造相同产品、使用相同方法，或已经作好制造、使用的必要准备
- 仅在原有范围内继续制造、使用，不构成侵权
- 证据要求：研发记录、采购订单、生产计划、内部评审文件等

#### 5.3 权利用尽（《专利法》第75条第1项）
- 若产品中的专利部件系从专利权人或其授权方合法购得

#### 5.4 科研及医药审批例外（《专利法》第75条第4、5项）
- 专为科学研究和实验而使用专利
- 为提供行政审批所需要的信息而制造、使用、进口专利药品或医疗器械

#### 5.5 规避设计方向

若侵权风险高且无有效抗辩，提出规避设计建议：
- **删除非必要特征**：若某特征为从属权利要求限定，尝试删除该特征是否仍能满足技术效果
- **替换等同特征**：用不同的技术手段实现相同功能，跳出等同原则范围
- **改变技术问题**：重新界定技术问题，使技术方案的出发点与专利不同
- **绕开功能性限定**：若专利使用功能性限定，查看说明书实施方式，设计完全不同的实现路径
- **设计-around 文档化**：规避设计过程须形成完整研发记录，防止被认定为故意侵权（影响惩罚性赔偿）

## 特别注意事项

1. **未公开专利申请的风险**：FTO 检索只能覆盖已公开专利，无法发现申请日在前但尚未公开的抵触申请。对于高度竞争领域，建议在产品上市前 18 个月启动 FTO（发明专利通常 18 个月公开）。
2. **实用新型专利的特殊性**：实用新型未经实质审查，权利稳定性差。若被实用新型专利指控侵权，可同步启动无效宣告请求（成本低、周期短）。
3. **GUI 外观设计**：若产品涉及软件图形用户界面，注意 2014 年以后中国保护的 GUI 外观设计（洛迦诺分类 14-04）。
4. **方法专利的延伸保护**：对于方法专利，依据《专利法》第11条，不仅方法本身受保护，依照专利方法直接获得的产品也受保护（举证责任倒置，由侵权人证明其产品制造方法不同于专利方法）。
5. **惩罚性赔偿**：故意侵权且情节严重的，法院可判决 1-5 倍惩罚性赔偿（《专利法》第71条）。若 FTO 发现高风险专利后仍故意实施，可能被认定为故意侵权。
