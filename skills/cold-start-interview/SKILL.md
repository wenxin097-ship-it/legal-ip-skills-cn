# ip-legal-cn 冷启动面试

> 本技能用于生成可选的企业知识产权实践档案，不是其他技能的强制前置条件。
>
> 其他技能可按需读取该档案；档案缺失时，应基于用户当前提供的材料继续工作并明确假设。
>
> 预计耗时：15-20 分钟 | 输出密级：内部 | 建议环境：私有化部署

---

## 目标

通过结构化问答，为企业建立知识产权实践画像（Practice Profile），生成其他知识产权 skills 可按需读取的 `LEGAL_PROFILE.md` 文件。档案可提高重复任务的一致性，但不得替代本次任务的事实确认。

**保密要求**：面试过程本身涉及企业管理制度、审批权限、外部机构联系人等内部信息，建议在私有化部署环境中执行，输出文件密级至少为"内部"。

## 执行前准备

1. 确认用户为企业知识产权管理人员或法务人员，了解企业基本情况。
2. 若企业已有 `LEGAL_PROFILE.md`，本 skill 支持"增量更新"模式——读取现有文件，仅更新用户明确回答的部分，保留已有内容。
3. 准备企业以下信息（尽量收集，未回答项保留 `[待填写]`）：
   - 行业领域与技术方向
   - 知识产权组合规模
   - 职务发明与奖励报酬制度
   - 外部服务机构名录
   - 审批权限矩阵
   - 保密分级规则
   - 开源软件管理政策

## 输出生成

面试结束后，根据用户回答生成标准化的 `LEGAL_PROFILE.md` 文件，格式与项目模板一致，所有 `[待填写]` 替换为用户提供的答案。若用户某题未回答，保留 `[待填写]` 并标注"【面试未回答】"提示。

### 生成规则

1. **已有文件增量更新**：若检测到当前目录或指定路径已存在 `LEGAL_PROFILE.md`，读取现有内容，仅覆盖用户明确回答的字段，保留未涉及的内容和格式。
2. **默认值填充**：若用户未提供某些通用信息，使用以下行业默认值（须明确告知用户）：
   - 布局策略：防御型与进攻型结合
   - 维权优先级：专利 > 商业秘密 > 商标 > 著作权
   - 保密分级：五级（公开/内部/秘密/机密/绝密）
   - 费减：不设置默认比例或资格阈值，标记为 `[待核验]`，并要求查询 CNIPA 当期规则
3. **日期自动填充**：
   - 版本号：v1.0
   - 更新日期：当前日期
   - 下次复审：当前日期 + 3 个月

### 输出后动作

1. 将生成的 `LEGAL_PROFILE.md` 保存到用户指定目录；未指定时放在当前项目的 `docs/` 目录。
2. 告知用户：其他 skills 会在文件存在且与任务相关时读取该档案。
3. 建议用户每季度复审一次，特别是在组合规模、审批权限、外部机构联系人发生变动时。
4. 若用户为保密办/知识产权总经理，建议将 `LEGAL_PROFILE.md` 纳入企业保密文件管理范围。

## 特别注意事项

1. 本 skill 本身不处理具体法律任务，但收集的信息将直接影响后续所有 skill 的输出准确性。
2. 面试问题可根据企业类型动态调整：
   - 若为专利代理事务所，弱化"内部审批权限"问题，强化"客户信息管理"问题
   - 若为高校/科研院所，增加"职务科技成果混合所有制""科研人员奖励报酬"等问题
   - 若为跨境企业，增加"出口管制""技术进出口许可证"等问题
3. 输出文件中的外部服务机构联系人信息属于内部信息，须按企业保密规则管理。
