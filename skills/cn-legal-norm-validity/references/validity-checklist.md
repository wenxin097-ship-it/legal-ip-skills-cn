# 规范效力检查表

| 维度 | 核验问题 |
|:---|:---|
| 文本 | 是否取得正式完整文本，而非摘要或转载 |
| 状态 | 现行、已修改、已废止、失效或尚未施行 |
| 版本 | 行为发生日和分析日分别适用哪个版本 |
| 主体 | 制定机关是否有权限，规则针对何种主体 |
| 层级 | 法律、行政法规、司法解释、规章或规范性文件 |
| 地域 | 全国、地方、自贸区、行业或平台范围 |
| 事项 | 实体、程序、证据、行政管理或内部操作规则 |
| 冲突 | 上位法、特别法、新法和过渡条款如何处理 |
| 后果 | 违反后导致无效、可撤销、行政处罚或其他后果 |

## 输出标签

- `A 可直接适用`：文本、效力和范围明确。
- `B 需结合解释`：存在开放概念、例外或事实前提。
- `C 仅供参考`：案例、政策、答复或内部规则。
- `D 不应引用`：失效、版本错误、越权或与问题无关。
