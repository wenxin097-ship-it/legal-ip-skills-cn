# 知识产权侵权初筛（中国法·四权综合）

## 目标

对涉嫌侵犯商标权、专利权、著作权或商业秘密的行为进行初步法律分析，评估维权可行性、识别风险点、建议最优行动路径。

**性质声明**：本技能为内部决策初筛，不构成正式侵权判定意见。是否启动维权程序须按权利人的内部授权流程审批。

## 四权适用框架

| 权利类型 | 核心法律 | 侵权行为类型 | 维权途径 | 赔偿计算 |
|:---|:---|:---|:---|:---|
| **商标权** | 《商标法》第57、60、63条 | 7类侵权行为（相同/近似商标、销售侵权商品、伪造标识等） | 行政查处（市监局）、民事诉讼、刑事（假冒注册商标罪） | 实际损失/侵权获利/许可费倍数/法定赔偿（500万上限）/惩罚性赔偿（1-5倍） |
| **专利权** | 《专利法》第11、65、71条 | 制造、使用、许诺销售、销售、进口专利产品；使用专利方法 | 行政查处（市监局/知识产权局）、民事诉讼、刑事（假冒专利罪） | 同商标，法定赔偿上限500万 |
| **著作权** | 《著作权法》第52-54条 | 复制、发行、出租、展览、表演、放映、广播、信息网络传播、改编、翻译、汇编等 | 民事诉讼、行政投诉（版权局/文化执法）、刑事（侵犯著作权罪） | 同商标，法定赔偿上限500万 |
| **商业秘密** | 《反不正当竞争法》第10、22、39条 | 不正当获取；披露、使用或允许他人使用；违反保密义务；教唆、引诱或帮助侵权 | 民事诉讼、刑事（侵犯商业秘密罪）、行政投诉（市监局） | 实际损失/侵权获利；故意且情节严重可适用惩罚性赔偿；法定赔偿依现行法核验 |

## 执行步骤

### Step 1: 权利基础稳定性评估

维权前必须先评估自身权利是否稳固，避免"维权不成反被无效/撤销"。

#### 1.1 专利权稳定性

| 检查项 | 通过标准 | 风险提示 |
|:---|:---|:---|
| 专利有效性 | 已缴当年年费，法律状态"专利权维持" | 若未缴年费，专利权已终止 |
| 无效历史 | 无无效宣告请求，或曾被请求但维持有效 | 若有pending无效案，维权可能被中止 |
| 权属清晰 | 专利权人名称与维权主体一致，无权属纠纷 | 若存在共有专利，须全体共有人同意 |
| 保护范围合理性 | 独立权利要求未被过窄解释 | 若审查历史中大量限缩，可能影响侵权比对 |
| 文件披露完整性 | 申请文件无隐瞒关键现有技术 | 若存在欺诈获得授权，可能被无效 |

#### 1.2 商标权稳定性

| 检查项 | 通过标准 | 风险提示 |
|:---|:---|:---|
| 商标有效性 | 在注册有效期内，已续展（如适用） | 若被撤销（连续三年不使用），权利丧失 |
| 使用证据 | 有持续使用证据（销售、广告） | 被控方可反提"撤三"（撤销连续三年不使用） |
| 知名度 | 在相关公众中有一定知名度 | 知名度低可能影响赔偿额和跨类保护 |
| 权属清晰 | 注册人名称与维权主体一致 | 转让未完成的，须先完成转让登记 |

#### 1.3 著作权稳定性

| 检查项 | 通过标准 | 风险提示 |
|:---|:---|:---|
| 作品完成时间 | 早于被控侵权行为发生时间 | 须保留创作底稿、首次发表证据 |
| 权属证据 | 有著作权登记证书或创作过程证据 | 中国著作权自动产生，但登记证书是权属初步证据 |
| 独创性 | 作品具有一定独创性（非简单复制公有领域素材） | 若作品缺乏独创性，不受保护 |
| 软件著作权 | 有软件著作权登记证书 | 源代码比对是软件侵权核心证据 |

#### 1.4 商业秘密稳定性

| 检查项 | 通过标准 | 风险提示 |
|:---|:---|:---|
| 秘密性 | 信息不为公众所知悉 | 若信息可通过公开渠道轻易获得，不构成商业秘密 |
| 价值性 | 具有商业价值（能带来竞争优势或经济利益） | 已淘汰技术、无经济价值的信息不构成商业秘密 |
| 保密性 | 采取了合理保密措施（保密协议、访问控制、加密、标注密级） | 未采取保密措施，法院可能认定不构成商业秘密 |
| 权属清晰 | 信息归公司所有（员工职务成果、委托开发约定） | 若权属有争议，维权基础不稳 |
| 范围明确 | 商业秘密的范围和内容清晰界定 | 笼统主张"所有技术资料"难以获得支持 |

**特别重要**：《反不正当竞争法》第39条举证责任规则：
- 权利人提供初步证据，证明其对所主张信息采取了保密措施，且合理表明商业秘密被侵犯后，涉嫌侵权人承担相应反证责任。
- 权利人仍须明确商业秘密载体、范围、形成过程、商业价值及保密措施，不能仅以概括性名称主张保护。
- 应分别分析“信息是否构成商业秘密”和“是否实施侵权行为”两层举证责任，不得笼统表述为全面倒置。

### Step 2: 侵权行为初步比对

#### 2.1 商标侵权比对（《商标法》第57条）

**7类侵权行为**：
1. 未经商标注册人许可，在同一种商品上使用与其注册商标**相同**的商标
2. 未经商标注册人许可，在同一种商品上使用与其注册商标**近似**的商标，或者在类似商品上使用与其注册商标**相同或近似**的商标，**容易导致混淆**的
3. 销售侵犯注册商标专用权的商品
4. 伪造、擅自制造他人注册商标标识或销售伪造、擅自制造的注册商标标识
5. 未经商标注册人同意，更换其注册商标并将该更换商标的商品又投入市场（反向假冒）
6. 故意为侵犯他人商标专用权行为提供便利条件，帮助他人实施侵犯商标专用权行为
7. 给他人的注册商标专用权造成其他损害的

**比对要点**：
- 商品/服务是否相同或类似（《类似商品和服务区分表》+ 市场实际）
- 商标是否相同或近似（音、形、义）
- 是否容易导致混淆（考虑商标知名度、被控方主观恶意、实际混淆证据）
- 销售者合法来源抗辩（第64条第2款：销售者能证明合法取得且不知侵权，不承担赔偿责任，但须停止销售）

#### 2.2 专利侵权比对

详见 `$claim-chart-builder`（侵权比对模式）。

核心要点：
- 全面覆盖原则 + 等同原则
- 功能性特征须结合说明书解释
- 注意抗辩：现有技术抗辩、先用权、权利用尽、科研使用例外

#### 2.3 著作权侵权比对

**核心规则**：接触 + 实质性相似

| 比对维度 | 说明 |
|:---|:---|
| **接触可能性** | 被控方是否有机会接触作品？（如员工跳槽、公开发表、展会展示） |
| **实质性相似** | 两部作品在表达层面是否相似？（思想-表达二分法：只保护表达，不保护思想） |
| **软件侵权** | 源代码比对（相同/相似代码比例）、目标程序反编译比对、软件界面/流程相似性 |
| **排除情形** | 合理使用（《著作权法》第24条）、独立创作、公有领域素材 |

**软件侵权特别提示**：
- 源代码比对是最直接证据，但获取被控方源代码困难
- 替代方案：目标代码反编译比对、软件运行界面/功能/错误信息比对
- 注意开源软件合规抗辩：被控方可能主张其代码来源于开源项目

#### 2.4 商业秘密侵权比对

**侵权行为类型**（《反不正当竞争法》第10条）：
1. 以盗窃、贿赂、欺诈、电子侵入或其他不正当手段获取权利人的商业秘密
2. 披露、使用或允许他人使用以前项手段获取的商业秘密
3. 违反保密义务或违反权利人有关保守商业秘密的要求，披露、使用或允许他人使用其所掌握的商业秘密
4. 教唆、引诱、帮助他人违反保密义务或违反权利人有关保守商业秘密的要求，获取、披露、使用或允许他人使用权利人的商业秘密

**比对要点**：
- 被控方信息是否与商业秘密**实质相同**（不要求100%相同，核心内容一致即可）
- 被控方获取信息的**途径**是否不正当（跳槽员工带走、黑客入侵、商业间谍）
- 被控方是否有**独立研发**的合理抗辩（须证明研发过程、研发记录、研发人员）
- 被控方是否通过**反向工程**合法获得（须证明产品合法取得、拆解过程记录）

### Step 3: 维权路径与成本收益分析

| 维权途径 | 优势 | 劣势 | 适用场景 | 预计周期 | 预计成本 |
|:---|:---|:---|:---|:---|:---|
| **行政投诉（市监局/知识产权局）** | 速度快、成本低、可查封扣押 | 赔偿有限（调解为主）、对复杂技术问题判断能力有限 | 商标/专利侵权事实清楚、证据易获取 | 3-6个月 | 5-15万 |
| **民事诉讼** | 可获得全额赔偿（含惩罚性赔偿）、可行为保全 | 周期长、成本高、举证责任重 | 赔偿要求高、需要禁令、技术复杂 | 1-2年（一审） | 20-100万+ |
| **刑事报案** | 威慑力强、可追究个人责任 | 立案门槛高、不解决民事赔偿 | 侵权规模大、违法所得高、重复侵权 | 6-18个月 | 10-30万 |
| **警告函/律师函** | 成本低、可保留证据、促成和解 | 无强制力、可能打草惊蛇（商业秘密） | 侵权规模小、希望快速和解、维护客户关系 | 1-4周 | 0.5-5万 |
| **无效宣告/撤销** | 可从权利稳定性层面消除或缩小障碍 | 仅针对特定权利程序，不直接产生损害赔偿 | 被诉侵权时的防御手段、清除障碍权利 | 按案件和主管机关进度估算 | 官费及代理成本按当期标准核验 |
| **平台投诉** | 快速下架、成本低 | 不解决实体争议、可能被反通知恢复 | 电商/网络侵权 | 1-4周 | 0-2万 |

**赔偿估算**：
- **实际损失**：权利人因侵权所受损失（销量下降 × 单位利润）
- **侵权获利**：侵权人因侵权所获利益（侵权产品销量 × 单位利润）
- **许可费参照**：仅在存在真实、可比、正常履行的许可交易时采用，并说明可比性调整。
- **法定赔偿**：分别依据任务发生时有效的《商标法》《专利法》《著作权法》和《反不正当竞争法》核验，不跨权利类型直接套用同一上限。
- **惩罚性赔偿**：分别核验主观要件、情节严重程度、计算基数及倍数范围；现行《反不正当竞争法》第22条规定故意侵犯商业秘密且情节严重时，可在法定计算基数的一倍以上五倍以下确定。

**维权成本收益比分析**：
- 若预计赔偿 < 维权成本，考虑发函和解或放弃
- 若侵权方为重要客户/合作伙伴，考虑商业影响后再决策
- 若涉及核心技术/品牌形象，即使赔偿不高，也应维权（防御性诉讼）

### Step 4: 反向风险识别

维权前须评估对方可能的反击：

| 反向风险 | 适用场景 | 应对措施 |
|:---|:---|:---|
| **专利无效宣告** | 我方专利维权，对方反提无效 | 提前准备无效答辩材料、筛选高稳定性专利维权 |
| **商标撤三** | 我方商标维权，对方反提连续三年不使用撤销 | 准备使用证据、考虑补充注册防御商标 |
| **恶意诉讼/恶意投诉反诉** | 对方主张我方滥用知识产权 | 确保证据充分、避免明显不成立的指控 |
| **反垄断抗辩** | 我方拒绝许可或收取高额许可费 | 确保许可条件符合 FRAND（SEP 场景） |
| **权属纠纷** | 对方主张我方不享有权利（如职务发明纠纷） | 提前梳理权属文件、劳动合同、委托开发合同 |
| **商业秘密反向取证** | 商业秘密案件中，对方可能主张我方也侵犯其商业秘密 | 确保我方技术来源合法、研发记录完整 |

## 特别注意事项

1. **诉讼时效**：
   - 商标/专利/著作权侵权：3年（自权利人知道或应当知道侵权行为之日起）
   - 商业秘密侵权：3年，且侵权行为具有持续性
   - 持续侵权案件须区分停止侵害请求与损害赔偿期间，不得概括写成一律从侵权终了之日起算

2. **行为保全（禁令）**：
   - 诉前/诉中申请停止侵权行为
   - 专利/商标/著作权：依据《专利法》第72条、《民事诉讼法》第104条
   - 须提供担保（法院酌定金额）
   - 是否适用紧急裁定期限，须根据具体权利类型、申请事项和现行程序法条文核验

3. **证据保全**：
   - 诉前证据保全：向侵权行为地或被申请人住所地有管辖权的法院申请
   - 公证保全：购买侵权产品、网页保全、展会保全（最常见的取证方式）

4. **管辖选择**：
   - 知识产权民事案件：最高人民法院知识产权法庭（技术类二审）、北京/上海/广州/海南自贸港知识产权法院（一审）、各地中级法院知识产权法庭（一审）
   - 选择管辖的考量：权利人所在地（便利）、侵权地（便于取证）、被告住所地（便于执行）、法院专业水平（北京知产法院、上海知产法院、广知院经验丰富）

5. **商业秘密案件的特殊保密**：
   - 依据现行程序法、商业秘密司法解释及案件具体情况，申请不公开审理或采取保密措施
   - 证据材料可要求密封、限制查阅范围
   - 判决书中可对商业秘密具体内容作隐去处理
   - 维权过程中应采用秘密点清单、分级披露、阅卷限制和脱敏版本，降低二次泄露风险
