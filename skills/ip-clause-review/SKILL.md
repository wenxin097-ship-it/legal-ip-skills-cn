# 技术合同知识产权条款审查（中国法）

## 目标

对技术开发、转让、许可、咨询、服务合同中的知识产权相关条款进行深度审查，识别权属风险、许可范围瑕疵、转让程序缺陷、保证条款缺失、违约责任不对等及反垄断合规隐患，输出带修订建议的审查意见书。

## 适用范围

- 技术开发合同（委托开发、合作开发）
- 技术转让合同（专利申请权转让、专利权转让、技术秘密转让）
- 技术许可合同（专利实施许可、技术秘密使用许可、软件许可）
- 技术咨询/技术服务合同（含知识产权归属条款的）
- 产学研合作协议（高校/科研院所与企业）
- 政府科研项目合同（科技部/工信部/地方科技计划）
- 跨境技术交易（需叠加《技术进出口管理条例》审查）

## 核心法律依据

- 《民法典》第三编第二分编第二十章（技术合同，第843-887条）
- 《专利法》第6条（职务发明）、第10条（转让登记生效）、第12条（许可备案对抗）、第15条（奖励报酬）、第77条（单位处分权）
- 《专利法实施细则》第13条（离职1年规则）、第77-78条（发明人报酬标准）
- 《计算机软件保护条例》第9条（软件著作权归属）、第18-22条（许可/转让）
- 《技术进出口管理条例》（跨境技术交易）
- 《促进科技成果转化法》（产学研合作）
- 《反垄断法》+ 市场监管总局《关于知识产权领域的反垄断指南》（限制竞争条款）
- 《劳动合同法》第23-24条（竞业限制）

## 执行步骤

### Step 1: 合同类型识别与法律框架匹配

| 合同类型 | 核心法律框架 | 高风险条款 |
|:---|:---|:---|
| **委托开发** | 《民法典》第859条 | 权属约定不明（无约定时归受托方！）、后续改进归属 |
| **合作开发** | 《民法典》第860条 | 权属约定不明（无约定时共同申请）、一方放弃申请权后的免费实施权 |
| **技术转让** | 《民法典》第862-863条、《专利法》第10条 | 转让登记生效、保密义务、后续改进归属、从属专利 |
| **技术许可** | 《民法典》第865-866条、《专利法》第12条 | 许可类型定义（独占/排他/普通）、分许可、交叉许可、许可费支付 |
| **软件许可** | 《计算机软件保护条例》第18-22条 | 开源许可证混用、SaaS 是否构成发行、升级维护中的著作权归属 |
| **产学研合作** | 《促进科技成果转化法》第18-20条 | 职务科技成果归属、科研人员奖励报酬（不低于转化净收入50%）、国资监管 |
| **政府科研项目** | 项目申报指南+合同模板 | 知识产权归属（原则上项目承担单位）、国家免费实施权、保密义务 |

### Step 2: 知识产权权属条款审查（最高优先级）

**审查清单**：

#### 2.1 基础权属约定
- [ ] 合同是否明确约定知识产权归属？（禁止模糊表述如"双方协商确定""原则上归甲方"）
- [ ] 若约定归一方所有，另一方是否保留了必要的使用权（如背景知识产权的继续使用权）？
- [ ] 若约定共有，是否明确了共有方式（共同共有/按份共有）、行使方式（单方实施/一致同意/按份实施）、收益分配比例？
- [ ] 是否区分了"背景知识产权"（Background IP）与"前景知识产权"（Foreground IP）？

#### 2.2 职务发明/职务成果风险
- [ ] 若对方为高校/科研院所，是否涉及其研究人员的职务发明？（《专利法》第6条）
 - 高校职务发明原则上归高校，但2015年《促进科技成果转化法》改革后，可约定归科研人员或企业
 - 必须确认高校已履行内部审批程序（国资评估、公示等）
- [ ] 若涉及离职人员（从本公司离职未满1年），其参与的发明创造是否可能被认定为原单位职务发明？（《专利法实施细则》第13条）
- [ ] 合同是否要求对方保证其交付的成果不侵犯第三方知识产权？（权利瑕疵担保）

#### 2.3 软件著作权特殊风险
- [ ] 委托开发的软件，若无约定，著作权归受托方（《计算机软件保护条例》第11条与《民法典》第859条一致）——**必须明确约定归委托方或共有**
- [ ] 是否约定了源代码交付义务？交付时间节点？
- [ ] 若仅获得目标代码许可，是否约定了升级、维护、Bug 修复的源代码级支持？
- [ ] SaaS 模式下，用户数据与生成内容的归属是否明确？

### Step 3: 技术转让条款审查

**《专利法》第10条核心规则**：
> "转让专利申请权或者专利权的，当事人应当订立书面合同，并向国务院专利行政部门登记，由国务院专利行政部门予以公告。**专利申请权或者专利权的转让自登记之日起生效。**"

**审查要点**：
- [ ] 转让合同是否为书面形式？（必须书面）
- [ ] 是否约定了办理著录项目变更登记的时限和责任方？（通常为转让方配合，受让方主导）
- [ ] 转让价款是否分期支付？是否挂钩登记完成节点？
- [ ] 转让前是否存在许可第三方的情况？若有，是否披露？许可合同是否继续有效？（原则上继续有效，但可约定解除）
- [ ] 是否约定了转让方在转让后一定期限内不得从事竞争性研发？（限制条款，注意合理性）
- [ ] 涉外转让：是否涉及《技术进出口管理条例》管制技术？是否需要办理技术出口许可证或自由进出口技术合同登记？

### Step 4: 技术许可条款审查

#### 4.1 许可类型定义（必须严格符合中国法定义）

| 许可类型 | 中国法定义 | 被许可人权限 | 许可人权限 | 第三方权限 |
|:---|:---|:---|:---|:---|
| **独占许可** | 仅被许可人可实施，许可人自身也不得实施 | 可单独起诉侵权 | 不得实施，不得再许可 | 不得获得许可 |
| **排他许可** | 许可人和被许可人均可实施 | 可与许可人共同起诉，或在许可人不起诉时单独起诉 | 可自行实施，不得再许可 | 不得获得许可 |
| **普通许可** | 许可人可许可多家 | 通常不可单独起诉（除非合同明确约定或取得诉权转让） | 可自行实施，可再许可 | 可获得许可 |

**审查要点**：
- [ ] 合同使用的"独家许可""唯一许可"等表述是否与中国法独占/排他/普通定义一致？
- [ ] 是否约定了分许可（Sub-license）权限？分许可范围？
- [ ] 许可期限是否超过专利有效期？（超过部分无效）
- [ ] 许可地域范围是否明确（中国大陆/全球/特定省份）？
- [ ] 许可费计算方式：固定许可费/入门费+提成/纯提成？提成基数是销售额还是利润？是否设置最低许可费（Minimum Royalty）？
- [ ] 是否约定了专利无效时的处理？（许可费是否退还、合同是否终止）
- [ ] 许可合同是否办理备案？《专利法实施细则》第14条规定专利实施许可合同备案可对抗善意第三人

#### 4.2 开源软件混用审查
- [ ] 若被许可技术包含开源软件，是否披露？开源许可证类型？
- [ ] GPL/LGPL/AGPL 传染性是否会影响被许可方的商业软件？
- [ ] 是否要求被许可方在分许可/销售时遵守开源许可证义务？

### Step 5: 保证与担保条款审查

**必须包含的保证**：
- [ ] **权属保证**：转让方/许可方保证其对标的知识产权享有合法、完整、无争议的权利
- [ ] **不侵权保证**：标的知识产权的实施不侵犯任何第三方知识产权
- [ ] **无质押/查封保证**：标的知识产权未设定质押、未被查封、未被提起无效宣告
- [ ] **技术成熟度保证**：技术可达到合同约定的性能指标（技术开发合同）

**责任承担方式**：
- [ ] 是否约定了侵权第三方索赔时的通知义务和协助义务？
- [ ] 是否约定了赔偿上限？（注意：故意隐瞒权属瑕疵的赔偿上限约定可能被认定为无效）
- [ ] 是否约定了知识产权无效时的回购/退款机制？

### Step 6: 限制竞争与反垄断审查

**高风险条款（可能违反《反垄断法》及知识产权反垄断指南）**：

| 条款类型 | 风险等级 | 说明 |
|:---|:---|:---|
| **固定转售价格** | 🔴 极高 | 限制被许可人向第三方销售专利产品的价格 |
| **限制最低转售价格** | 🔴 极高 | 限制被许可人向第三方销售专利产品的最低价格 |
| **独占性回授** | 🟡 高 | 要求被许可人将后续改进技术独占性地回授给许可人 |
| **不质疑条款** | 🟡 高 | 禁止被许可人对标的知识产权的有效性提出异议 |
| **无正当理由搭售** | 🟡 高 | 要求被许可人在获得专利许可时必须同时购买非专利产品/接受非专利许可 |
| **限制技术来源** | 🟡 中 | 限制被许可人从第三方获得竞争性技术 |
| **限制生产数量/销售地域** | 🟢 低-中 | 在知识产权许可中，地域限制通常被允许；产量限制需有正当理由 |
| **竞业限制** | 🟡 中 | 限制期限超过2年、未支付补偿金、范围过宽，可能被认定无效 |

**审查输出**：对每条限制竞争条款，标注风险等级，并提供替代表述建议。

### Step 7: 违约责任与争议解决

- [ ] 违约金是否约定过高？（可请求法院调减，但会增加诉讼不确定性）
- [ ] 是否约定了知识产权被无效/撤销时的合同终止与清算机制？
- [ ] 争议解决方式：诉讼 vs 仲裁？
  - **建议**：涉及专利有效性的纠纷，建议约定法院诉讼（中国仲裁庭无权宣告专利无效）
  - 若约定仲裁，须明确仲裁机构（北仲/上仲/贸仲等），并约定仲裁地
- [ ] 管辖法院：是否约定知识产权法院/法庭管辖？（北京/上海/广州/海南自由贸易港/最高人民法院知识产权法庭）
- [ ] 法律适用：涉外合同是否约定适用中国法？（技术进出口合同强制适用中国法，《民法典》第467条）

## 特别注意事项

1. **政府科研项目特殊规则**：
   - 《国家科技计划项目管理暂行办法》通常约定知识产权归项目承担单位，但国家保留"为公共利益目的"的免费实施权
   - 若项目涉及国家秘密，须叠加《保守国家秘密法》《科学技术保密规定》审查
   - 财政资助项目形成的技术成果，完成人享有优先受让权（《促进科技成果转化法》第18条）

2. **高校/科研院所国资监管**：
   - 高校科技成果转让、许可、作价入股，原则上须履行资产评估、公示程序（《事业单位国有资产管理暂行办法》）
   - 但《促进科技成果转化法》第18条允许高校自主决定转化方式（除涉及国家秘密、国家安全外），需注意地方/高校细则差异

3. **跨境技术交易**：
   - 限制类技术出口：须办理技术出口许可意向书 + 技术出口许可证
   - 自由类技术出口：须办理技术出口合同登记证
   - 禁止出口的技术：不得出口（参见《中国禁止出口限制出口技术目录》）
   - 技术进口：自由类技术进口须办理合同登记证；限制类须办理许可证

4. **软件著作权登记的对抗效力**：
   - 软件转让/许可不以登记为生效要件，但登记可作为对抗第三人的初步证据（《计算机软件著作权登记办法》）
   - 建议在合同中约定办理软件著作权登记/变更登记的义务

5. **标准必要专利（SEP）许可**：
   - 若合同涉及 SEP 许可，须审查 FRAND 承诺（公平、合理、无歧视）
   - 注意最高法《关于审理标准必要专利纠纷案件适用法律若干问题的规定》对禁令、费率、反垄断的特殊规则
