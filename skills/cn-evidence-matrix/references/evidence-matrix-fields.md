# 证据矩阵字段

| 字段 | 说明 |
|:---|:---|
| 主张/抗辩 | 需要支持的法律立场 |
| 构成要件 | 对应规范要件 |
| 待证事实 | 必须证明的具体事实 |
| 证明责任 | 由哪一方承担 |
| 证据编号 | 唯一编号 |
| 证据名称 | 文件、实物、电子数据、证言等 |
| 来源与持有人 | 形成主体和当前控制人 |
| 形成日期 | 原始形成时间 |
| 取得方式 | 自有、公开、调查、保全等 |
| 原件状态 | 原件、复制件、截图、导出件 |
| 真实性 | 支持点和质疑点 |
| 合法性 | 取得和使用是否合法 |
| 关联性 | 与待证事实的连接 |
| 证明力 | 强、中、弱及理由 |
| 反证 | 对方材料或替代解释 |
| 缺口 | 尚缺材料及影响 |
| 下一步 | 补证、保全、鉴定、调查令等 |
