# 事实与争点字段

## 主体表

| 字段 | 内容 |
|:---|:---|
| 主体 | 法定名称或匿名标识 |
| 身份 | 当事人、代理人、员工、平台、监管机关等 |
| 关系 | 合同、劳动、授权、控制或竞争关系 |
| 权限 | 签约、处分、代理或审批权限 |
| 证据 | 身份和权限的证明材料 |

## 事实状态

- `F1 已证实`：有可靠材料支持且无实质争议。
- `F2 有证据待核`：已有材料但真实性或完整性待核。
- `F3 单方陈述`：仅一方主张。
- `F4 推断`：由其他事实推导。
- `F5 缺失`：结论所需但尚无信息。

## 争点写法

使用“在事实 X 和规范 Y 下，主体 A 是否满足要件 Z”。
避免使用“谁对谁错”“是否违法”这类无法直接检索和证明的宽泛问题。
